# LLM Engineering Projects
This repository contains a collection of applied projects focused on Large Language Model (LLM) engineering.

These are personal, exploratory projects used to test ideas, architectures, and design patterns for building LLM-based systems. The emphasis is on developing practical components such as chatbot assistants, tool-augmented workflows, and domain-specific applications.

The goal is to evaluate how LLMs can be structured, constrained, and integrated into real-world workflows such as pricing systems, information retrieval, and decision support. Insights from these experiments are intended to be refined and later integrated into production-level applications.
